# LG Electronics CRM & Cotizador
**Ubicación del Proyecto:** `d:\Programas\Antigravity\1`

## Descripción
Sistema de gestión de relaciones con clientes (CRM) y generador de cotizaciones para LG Electronics Eco Solutions.

### Módulos Principales:
- **Dashboard:** Vista general de KPIs y métricas.
- **Cotizador:** Herramienta para crear y gestionar cotizaciones con cálculo de precios y descuentos.
- **Aprobaciones:** Flujo de trabajo para aprobar descuentos especiales.
- **Reportes:** Exportación de datos a Excel.

## Tecnologías
- HTML5, CSS3, JavaScript (Vanilla)
- Supabase (Base de datos y Autenticación)
- Chart.js (Gráficos)
- SheetJS (Exportación Excel)
- jsPDF (Generación de PDF)
